Description

Function (main function) entry point of module

Principle Behind

When the Python interpreter reads a source file, it executes all of the code found in it.

Before executing the code, it will define a few special variables.

  • If Python is running the file as the main program then Python will create a __name__ variable with the value of main .

  • If python is importing the file as an import into an already running main the __name__ variable will be set to the moduleâ€™s name in the modules scope .



Basic structure

if __name__ == '__main__':
    STATEMENT_1
    STATEMENT_2
    ....
